## Ход работы

### 1. Аналитическое решение

**Исходное уравнение:**
$$a\ddot{x} + b\dot{x} + c x = d$$
где $a = -8.77$, $b = -9.69$, $c = -1.99$, $d = -2.85$

**В характеристическом виде:**
$$-8.77r^2 - 9.69r - 1.99 = 0$$

**Дискриминант:**
$$D = b^2 - 4ac = 93.8961 - 69.8092 = 24.0869$$
$$r_{1,2} = \frac{-b \pm \sqrt{D}}{2a}$$
$$r_1 = -0.832, \quad r_2 = -0.273$$

**Однородное решение:**
$$x_h(t) = C_1 e^{-0.832t} + C_2 e^{-0.273t}$$

**Частное решение:**
$$c x_p = d$$
$$x_p = \frac{d}{c} = \frac{-2.85}{-1.99} = 1.432$$

**Общее решение:**
$$x(t) = C_1 e^{-0.832t} + C_2 e^{-0.273t} + 1.432$$

**Зададим начальные условия** $x(0) = 0.1$, $\dot{x}(0) = 0$:
$$C_1 + C_2 + 1.432 = 0.1$$
$$-0.832 C_1 - 0.273 C_2 = 0$$

Решая систему уравнений:
$$C_1 = -0.032$$
$$C_2 = -1.3$$

**Итоговое решение:**
$$x(t) = -0.032 e^{-0.832t} - 1.3 e^{-0.273t} + 1.432$$

### 2. Решение численными методами

Зададим $y_1 = x$, $y_2 = \dot{x}$. Тогда:
$$\dot{y_1} = y_2, \quad \dot{y_2} = \frac{d - c y_1 - b y_2}{a}$$

#### 2.1. Явный метод Эйлера
$$y_i = y_{i-1} + h f(t_{i-1}, y_{i-1})$$

#### 2.2. Неявный метод Эйлера
$$y_i = y_{i-1} + h f(t_i, y_i)$$

#### 2.3. Метод Рунге-Кутта 4-го порядка
$$k_1 = f(t_{i-1}, y_{i-1})$$
$$k_2 = f(t_{i-1} + \frac{h}{2}, y_{i-1} + \frac{h}{2}k_1)$$
$$k_3 = f(t_{i-1} + \frac{h}{2}, y_{i-1} + \frac{h}{2}k_2)$$
$$k_4 = f(t_{i-1} + h, y_{i-1} + h k_3)$$
$$y_i = y_{i-1} + \frac{h}{6}(k_1 + 2k_2 + 2k_3 + k_4)$$
